# Desarrollo de Aplicaciones Web con ASP.NET WebForms

Este proyecto representa mi transición del desarrollo de escritorio al mundo web, aplicando los fundamentos de **C#** y **Programación Orientada a Objetos (POO)** en la arquitectura de Microsoft .NET.

## Características
- Implementación de controles de servidor ASP.NET.
- Manejo del ciclo de vida de una página WebForm (Page Life Cycle).
- Persistencia de datos y navegación entre páginas.
- Uso de Master Pages para un diseño consistente.

## Tecnologías utilizadas
- **Lenguaje:** C#
- **Framework:** ASP.NET WebForms (.NET Framework)
- **Frontend:** HTML5, CSS3, y componentes de servidor.
- **Lógica de Negocio:** POO avanzada heredada de niveles anteriores.

## Estructura del Proyecto
- `Capa Dominio:` Clases y entidades del negocio.
- `Capa Negocio:` Lógica de procesamiento de datos.
- `Capa Web:` Interfaz de usuario y archivos `.aspx`.
